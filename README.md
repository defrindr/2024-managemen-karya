## Deploy

1. Clone this repository

```sh
git clone https://github.com/defrindr/managemen-karya-2024.git
```

2. Install dependecies

```sh
composer install
```

3. Copy .env & setup environment

4. Generate key

```sh
php artisan key:generate
```

5. Running migration

```sh
php artisan migrate:fresh --seed
```

6. Ruuning server

```sh
php artisan serve
```
